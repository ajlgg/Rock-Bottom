# Rock Bottom

**Authors:** AJ Luthra, Owen Fazzani, Timothy Jeon  

---

## Introduction

A true disruptor in its field, **Rock Bottom** aims to revolutionize the media ecosystem by removing abstractions from the overstimulating nature of modern short-form video entertainment. It provides users with a direct, high-engagement experience designed to maximize retention and interaction.

By cultivating a highly engaged user base, Rock Bottom positions itself to compete with industry leaders such as Instagram and TikTok.

---

## Gambling Features

Rock Bottom introduces a unique feature set not commonly found in short-form video platforms: integrated casino-style games.

### Thrill Mode

Users can activate **Thrill Mode**, allowing them to gamble in-app currency (**RBTokens**) on various games:

- Roulette  
- Blackjack  
- Poker  

### Game Details

- **Blackjack**
  - Played in lobbies (up to 6 users)
  - Users play against a computer dealer  

- **Poker**
  - Multiplayer-only (human vs. human)
  - Hosted in shared lobbies (up to 6 users)  

### Sophistication System

Winning RBTokens increases a hidden user metric called **Sophistication**.

- Purely aesthetic (not visible to users)
- Alters background music in Thrill Mode:
  - Low: Unlicensed Top 40-style music  
  - Mid: Rock and rap variations  
  - High: Classical and opera music

## Scrolling Features

The core experience revolves around a continuous short-form video feed.

### Core Mechanics

- Infinite scrolling using mouse wheel  
- Videos are capped at **30 seconds**  
- Users earn:
  - **+10 XP**
  - **+1 RBToken**
  - Condition: Watch a video for at least 15–30 seconds  

### Progression System

- Level progress bar (top-right UI)
- Displays:
  - Current level  
  - Level name  

---

## Content Creation

Users can create and share their own videos within the platform.

### Features

- Record videos directly in-app  
- Publish with:
  - Title  
  - Description  

---

## UI Features

- Fade-in / fade-out emojis on multi-click  
- Save video button  
- (+) button for recording  
- Simple, minimal scrolling interface  

---

## Summary

Rock Bottom blends addictive short-form content with gamification and gambling mechanics to create a highly engaging and differentiated user experience.
